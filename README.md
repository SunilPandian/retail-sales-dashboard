# 📊 Retail Sales Performance Dashboard

## 🔍 Project Overview

This project analyzes retail sales data using SQL and Power BI to uncover business insights related to revenue trends, customer behavior, and product performance.

## 🛠 Tools Used

* SQL (MySQL) – Data cleaning & transformation
* Power BI – Data visualization & dashboarding

## 📈 Key Insights

* Revenue shows a declining trend with a sharp drop in May
* February had higher order volume but lower average order value
* March showed fewer but high-value transactions
* Majority of revenue is concentrated in Europe

## 📊 Dashboard Features

* KPI Cards (Revenue, Orders, Customers, AOV)
* Monthly Revenue Trend
* Average Order Value Analysis
* Top 10 Products by Revenue
* Country-wise Revenue Map
* Interactive Country Filter

## 📂 Files Included

* Power BI Dashboard (.pbix)
* Dataset (CSV)

## 🚀 How to Use

Download the `.pbix` file and open it in Power BI Desktop to explore the dashboard.
